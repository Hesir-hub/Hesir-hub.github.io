发表于2025-06-10
复试记账法复试记账法，也叫借贷记账法，需要遵从一个会计原则和三个会计恒等式 会计原则：有借必有贷，借贷必相等 资产 = 负债 + 所有者权益 利润 = 收入 - 支出 资产 + 支出 = 负债 + 所有者权益 + 收入其中，（资产 + 支出）表示借记账户（Debit Account），反应资金去脉；（负债 + 所有者权益 + 收入）表示贷记账户（Credit Account），反应资金来龙 资金流分为虚拟资金流和实体资金流 虚拟资金流：描述内部创建的账户的资金流动 实体资金流：在银行有具体的账户，描述这些账户的资金流动 虚拟资金流表示内部账户直接的资金流动，箭头指向不代表先后顺序，永远从借记账户指向贷记账户，表示借贷关系实体资金流表示银行账户之间的资金流动
支付基础知识
发表于2025-06-10
拒付信用卡五要素在境外，信用卡除了通过POS机刷卡支付、绑卡支付外，还可以通过信用卡五要素进行支付 卡号 持卡人姓名 生效时间 失效时间 CVV码（卡片背后的3位数校验码） 拒付流程 持卡人向发卡行提交拒付申请 发卡行向卡组织提交拒付申请 卡组织向收单机构发起拒付通知 收单机构向商户发起拒付通知 商户向收单机构拒付回应（完全接收拒付、部分接受拒付、不接受拒付） 收单机构通知卡组织 卡组织通知发卡行 发卡行裁定拒付结果 发卡行通知拒付结果如果不接受拒付，商户需要提交拒付抗辩材料，由发卡行进行拒付判责拒付反转：拒付抗辩成功，收单机构会退还资金给商户拒付仲裁：如果第二轮拒付抗辩失败，商户可以提起拒付仲裁，仲裁由卡组织负责 拒付和退款的差别 退款：消费者联系商户 拒付：消费者联系发卡行
